# dynamic-to-do-list-js

## To-Do List App

A simple To-Do List application built with HTML, CSS, and JavaScript.

## Features

- Add tasks via input field and Enter key or Add Task button
- Remove tasks using the Remove button
- Prevents empty tasks
- Dynamic task list with clean interface

## Usage

- Open index.html in a browser
- Type a task and press Enter or click Add Task
- Remove tasks by clicking Remove

## File Structure

- index.html
- styles.css
- script.js
- README.md

## Future Enhancements
